# Install And Start Dev

Install deps

```
Yarn install
```

Install Commitizen

```
npm install -g commitizen
```

Start dev

```
yarn dev
```

# Commit

Check your branch

```
git checkout -b feat-xxx
```

Use commitizen

```
git cz
```
